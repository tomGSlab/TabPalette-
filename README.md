## ✨ Philosophy & Overview

**TabPaletté（タブパレット）** は、ブラウザを開くたびに美しさとインスピレーションを提供する Chrome 拡張機能です。
深い漆黒の背景に、ブックマーク（グループ）と付箋（Sticky Notes）をタイルのように配置。ノイズを徹底的に排除したミニマルなUIが、情報へのアクセスをかつてなくスマートにします。

## 💎 Core Features

### 🍱 Bento Grid Layout

情報を美しいグリッドへ。直感的なドラッグ＆ドロップによる自由なレイアウト構築が可能です。
絶対的な位置を保持したい要素には「ピン留め」を。流動性と固定性が美しく共存するワークスペースを実現します。

### 🪄 Smart Tidy Up

ワンクリックで、カオスから完全な秩序へ。
「左側にブックマークグループ、右側に付箋」という洗練された配置へ、ドラマチックに自動整列します。ピン留めされた要素は周囲の動きに惑わされることなく、グリッドにスナップしたまま現在の定位置を死守します。

### 🌊 Smooth Interaction

すべての操作に、極上の手触りを。
グループ、付箋、あらゆる要素の移動に滑らかなアニメーションを実装。思考を妨げない、シームレスで心地よいインタラクションを提供します。

### 🎨 Custom UI

神は細部に宿る。
ブラウザ標準の野暮ったいUI要素は一切排除。独自実装によるスタイリッシュなカスタムドロップダウン、視認性の高い洗練されたキャレットなど、1pxの妥協も許さないデザイン哲学が貫かれています。

### 🔒 Privacy First

あなたのデータは、あなただけのローカルな世界に。
外部サーバーとの通信は一切なく、`chrome.storage.sync` を用いてブラウザの同期ストレージ内に安全に保存されます。

## 🚀 Installation

開発者モードを利用して、数ステップで導入可能です。

1. Zipファイルをダウンロードし、ディレクトリを展開します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理ページへアクセスします。
3. 画面右上の **「デベロッパーモード」** をオンに切り替えます。
4. 画面左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックし、展開した `TabPaletté` のディレクトリを選択します。

新しいタブを開き、あなただけのキャンバスを描き始めてください。

## 📖 Usage & Workflow

1. **Create & Paint**
   - 画面上のコントロールから、ブックマークを束ねる「グループ」や、手軽なメモを残す「付箋」を追加します。
2. **Arrange (Bento Grid)**
   - 追加した要素をドラッグして、自分にとって最適なレイアウトを探求します。
3. **Pin & Lock**
   - 常に特定の場所に置いておきたいコアな情報は「ピン留めアイコン」をクリックして位置をロックします。
4. **Smart Tidy Up (高度な整理)**
   - 画面が雑然としてきたら、Tidy Up ボタンを実行。ピン留め要素の位置は死守したまま、残りの要素が「左側にグループ・右側に付箋」というルールの下で美しいBentoレイアウトへと再構築されます。
